# Tic Tac Toe AI

A terminal-based Tic Tac Toe game built with Python where players compete against an AI opponent.

## Features

- Single Player Mode
- AI Opponent
- Win Detection
- Draw Detection
- Replay Option
- Input Validation

## Technologies Used

- Python
- Random Module

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/tic-tac-toe-ai.git
```

Move into the project directory:

```bash
cd tic-tac-toe-ai
```

Run the game:

```bash
python tic_tac_toe.py
```

## Gameplay

- You play as X
- AI plays as O
- First player to align three symbols wins
- Positions are numbered 1-9

Example:

1 | 2 | 3
--|---|--
4 | 5 | 6
--|---|--
7 | 8 | 9

## AI Logic

The AI follows these rules:

1. Win if possible
2. Block player wins
3. Take center square
4. Choose a random available move
